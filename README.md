## Node.js version

- The current app runs on Node.js from `server.js`.
- Lobby and room are split into two pages: `/lobby.html` and `/room.html`.
- The root route `/` opens the lobby page.
- Use `node server.js` locally or configure a Node.js app in Plesk.
- Keep `public/runtime-config.js` in sync if frontend and API use different domains.

# XO Classic Online

Онлайн-игра в крестики-нолики с нестандартным правилом: у каждого игрока на доске может быть только 3 фигуры одновременно. Когда игрок ставит четвёртую фигуру, самая старая его фигура исчезает.

## Что умеет

- Создание комнаты с ником игрока
- Подключение второго игрока по коду комнаты или по прямой ссылке
- Настройка подсказки перед стартом: подсвечивать фигуру, которая исчезнет следующей
- Синхронизация игры по сети в реальном времени
- Автоматическое восстановление своей сессии после перезагрузки страницы

## Запуск

```bash
python server.py
```

После запуска сайт будет доступен по адресу:

```text
http://localhost:8000
```

## Запуск по локальной сети

Можно использовать готовый PowerShell-скрипт:

```powershell
.\start_lan_server.ps1
```

Он покажет адреса вида `http://172.20.139.126:8000`, которые можно открыть с другого компьютера в той же сети.

Если открываешь сайт на этом компьютере через `localhost`, кнопка копирования ссылки в комнате всё равно будет отдавать LAN-ссылку для второго игрока.
